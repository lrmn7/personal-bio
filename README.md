## Ussage

```sh
git clone https://github.com/lrmn7/personal-bio.git
```
- Install the dependencies

```
npm install # for npm
yarn install # for yarn
pnpm install # for pnpm
```

- Start the server

```
npm run dev # for npm
yarn dev # for yarn
pnpm dev # for pnpm
```
